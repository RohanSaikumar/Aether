# 🌙 Aether

A full-stack AI assistant built using the MERN stack with long-term semantic memory powered by RAG and MongoDB Atlas Vector Search.

## 🌐 Live Demo

🔗 Frontend: https://aether-eta-ivory.vercel.app

🔗 Backend API: https://aether-backend-lmhq.onrender.com

## ✨ Features

- 🔐 User Authentication (Passport.js + Sessions)
- 🔒 Password Hashing (bcrypt)
- 🧠 Long-Term Memory using RAG
- 📚 MongoDB Atlas Vector Search
- 💬 Multi-thread Conversations
- 👤 Multi-user Support
- 🤖 OpenAI Integration
- ⚡ React Frontend
- 🌐 Express Backend

## 🛠️ Tech Stack

- React
- Node.js
- Express
- MongoDB Atlas
- Passport.js
- OpenAI API
- bcrypt
